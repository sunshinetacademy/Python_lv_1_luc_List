في المحاضرتين الأوليين، تعلمنا إن الـ For Loop بيكرر كود لعدد محدد من المرات.

for i in range(5): print(i)

النتيجة: 0, 1, 2, 3, 4

📿 مع النصوص

for ch in "HI": print(ch)

🎁 ما هي الـ List؟

الـ List هي صندوق سحري يقدر يخزن كتير من القيم في مكان واحد!

📦✨

🔢 الوصول للعناصر (Indexing)

كل عنصر في الـ List ليه رقم خاص (Index) نستخدمه عشان نوصله.

0🍎
1🍌
2🍊
3🍇
4🍓

📝 الشكل:

fruits = ["🍎", "🍌", "🍊", "🍇", "🍓"] print(fruits[0]) # أول عنصر print(fruits[2]) # العنصر الثالث print(fruits[4]) # آخر عنصر
⚠️ فخ مهم جداً:
  • 🎯 الترقيم بيبدأ من 0 مش 1!
  • 🎯 العنصر الأول = [0]
  • 🎯 العنصر التاني = [1]
  • 🎯 لو كتبت رقم كبير → Error!

💡 Lists مش زي النصوص:

📏 طول الـ List: len()

دالة len() بتقولك كم عنصر في الـ List.

📝 أمثلة:

fruits = ["🍎", "🍌", "🍊", "🍇"] print(len(fruits)) # 4 empty = [] print(len(empty)) # 0
🌟 ليه مهمة؟
عشان نعرف آخر عنصر: list[len(list)-1]
أو ببساطة: list[-1] 🎯

🔗 ربط مع For Loop:

scores = [85, 92, 78, 95, 88] # عدد العناصر print("عدد الدرجات:", len(scores)) # استخدام في for loop for i in range(len(scores)): print(f"Student {i}: {scores[i]}")

➕ إضافة عناصر: append()

دالة append() بتضيف عنصر جديد في آخر الـ List.

list.append(value)

بتضيف value في آخر الـ List.

📝 مثال:

fruits = ["🍎", "🍌"] fruits.append("🍊") fruits.append("🍇") print(fruits) # ["🍎", "🍌", "🍊", "🍇"]

➖ حذف العناصر

عندنا طريقتين لحذف العناصر من الـ List.

list.remove(value)

بتحذف أول عنصر قيمته مطابقة.

fruits = ["🍎", "🍌", "🍎"] fruits.remove("🍎") # ["🍌", "🍎"]
list.pop(index)

بتحذف العنصر في index وترجعه.

fruits = ["🍎", "🍌", "🍊"] fruits.pop(1) # ["🍎", "🍊"]

🎓 اختبار الفهم الشامل

8 أسئلة لقياس فهمك الكامل!

❓ السؤال 1: كيف ننشئ List؟

أ) fruits = ("🍎", "🍌")
ب) fruits = ["🍎", "🍌"]
ج) fruits = {"🍎", "🍌"}
د) fruits = <"🍎", "🍌">

❓ السؤال 2: ما ناتج fruits[1]؟

fruits = ["🍎", "🍌", "🍊"]
أ) 🍎
ب) 🍌
ج) 🍊
د) Error

❓ السؤال 3: كيف نضيف عنصر؟

أ) list.append(value)
ب) list.add(value)
ج) list.push(value)
د) list + value

❓ السؤال 4: ما ناتج len([1,2,3,4])؟

أ) 3
ب) 4
ج) 5
د) Error

❓ السؤال 5: ماذا تفعل fruits[-1]؟

أ) ترجع أول عنصر
ب) ترجع آخر عنصر
ج) تحذف آخر عنصر
د) Error

❓ السؤال 6: كيف نحذف "🍌" من القائمة؟

أ) fruits.delete("🍌")
ب) fruits.remove("🍌")
ج) fruits - "🍌"
د) delete fruits["🍌"]

❓ السؤال 7: ما ناتج الكود؟

nums = [1, 2, 3] for n in nums: print(n * 2)
أ) 1, 2, 3
ب) 2, 4, 6
ج) [2, 4, 6]
د) Error

❓ السؤال 8: إمتى نستخدم While مع List؟

أ) لما نعرف عدد العناصر
ب) لما نكمل بشرط معين
ج) دايمًا
